## 数组取值语法与数组指针

- 符号`*`代表从指针里面取值
- 符号`&`代表从值里面去指针地址
- 数组变量“arr”自身是指向第一个元素的指针

数组在内存里空间地址是连续的，可以通过(指针+1)的方式来取得下一位元素，为了验证这个说法，我们来做一个实验

```c

    // *(arr) 相当于从内存中取出 arr[0] 内容为 {0,1,2}
    // *(arr+1) 相当于从内存中取出 arr[1] 内容为 {10,11,12}
    
    int *subArr1 = *(arr+1);
	printf("%d : %d",subArr1[0],arr[1][0]);//输出 10 : 10
    
	
	
	
```
