编译静态文件优化搜索排名-docsify编译工具

docsify官网

docsify简易入门

工具源码仓库

将docsify文档每个页面编译成独立的html文件

安装

npm i docsify-build -g

or

yarn global add docsify-build

使用

docsify-build -d ./docs -o ./dist

参数列表

--o 输出路径 默认 ./dist
--d 文档路径 默认 ./
--t html模版路径
--c 配置文件路径

配置文件示例

module.exports =  {
    name: 'name',
    repo: 'github/repository',
    routerMode: 'history',
    loadSidebar: true,
    auto2top: true
}
