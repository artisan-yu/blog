# 属性域&lt;w:*Pr&gt;&lt;/w:*Pr&gt;

- `&lt;w:*Pr&gt;&lt;/w:*Pr&gt;`

## 段落属性

```
<w:p>
    <!--段落的属性域 paragraph properties-->
    <w:pPr>
        <!-- 居中对其 -->
        <w:jc w:val="center">
    </w:pPr>
    <w:t>你好</w:t>
</w:p>
```

## 行内局部属性

```
<w:p>
    <w:r>
        <!--run标签的样式-->
        <w:rPr>
            <!-- 粗体 -->
            <w:b/>
        </w:rPr>
        <w:t>你</w:t>
    </w:r>
    <w:r>
        <w:t>好</w:t>
    </w:r>
</w:p>
```
