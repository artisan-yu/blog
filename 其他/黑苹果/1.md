# OC部分ACPI描述

https://dortania.github.io/Getting-Started-With-ACPI/


| 名称                   | 描述                                                      |
|----------------------|---------------------------------------------------------|
| SSDT-PLUG或SSDT-PLUG-DRTNIA | CPU的电源管理                                                |
| SSDT-EC-USBX或SSDT-EC | 提供USB电源属性<br/>用于热键和电池<br/>(Skylake及以上架构用的是SSDT-EC-USBX) |
| SSDT-AWAC            | 修复系统时钟                                                  |
| SSDT-RHUB            | 修复usb                                                   |
| SSDT-PNLF            | 修复笔记本背光                                                 |
| SSDT-XOSI或SSDT-GPI0  | 修复触控板                                                   |
